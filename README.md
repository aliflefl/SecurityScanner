# تطبيق ماسح الأمان (SecurityScanner)

تطبيق أندرويد متقدم لفحص الثغرات الأمنية ومحاكاة اختبارات الاختراق.

## الميزات الرئيسية
- مسح شبكات WiFi
- كشف الثغرات الأمنية
- فحص جدران الحماية
- واجهة مستخدم عربية كاملة
- نتائج فورية مع محاكاة حقيقية
